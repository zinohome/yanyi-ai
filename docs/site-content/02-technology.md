### 底座 + 三引擎

_大脑、记忆、感官各司其职，由 IndustrialX 底座编排；每个能力独立、可组合、可替换。_

- **大脑 · CozyEngine**（思考与语言编排） — 统一意图理解、任务规划、记忆调度与工具路由决策；作为唯一会话事实源，把工具执行委托给底座的执行层。

- **记忆 · CozyMemory**（统一记忆平台） — 一套 API 整合对话记忆、用户画像与知识图谱；权威知识（合规、规格）进 RAG/KB 层并人工策展，低可信信息进 Memory 层，两者信任级别不混淆。

- **感官 · RTVoice**（实时语音底座） — 实时语音转写、流式合成与端到端语音对话，全栈本地推理，单 GPU 即可运行；通过标准 API 接入应用，敏感语音数据无需出网。

### 核心技术能力

_围绕真实企业环境的可靠落地_

- **工业文档解析** — 工程图纸、FA 报告、ECN/PCN、BOM、合规证书等脏数据的硬化解析

- **知识结构化与图谱** — 围绕产品、失效模式、变更影响建立可复用知识网络

- **智能工作流编排** — 持久化编排跨天跨周、可审计、不丢状态的业务流程

- **私有化部署** — 完全内网运行、可离线，数据不出厂

- **本地小模型推理** — 靠结构而非模型规模拿到工业级可靠性，降低算力门槛

- **可靠可审计输出** — 抽取式 + 可追溯 + 查不到就转人工，宁可弃权不猜

### 私有化优先，安全可控

面向制造企业对数据安全、知识产权与内部流程的高要求，IndustrialX 支持私有化部署与离线运行，敏感数据不出厂，核心知识资产留在客户内网环境中。

系统输出过程可留痕、可追溯、可审计，适用于对可靠性、安全性与合规性要求较高的企业级场景；License 清晰，可进企业尽调、可白标交付。
